# 🎓 Pro AI Assignment Writer

A high-fidelity, free, and unlimited academic writing assistant. Generate research papers, essays, case studies, and more with professional formatting and citations.

![Preview](https://raw.githubusercontent.com/imfj007/assignment-writer-tool-/main/preview.png) *(Note: Add your own preview screenshot here!)*

## ✨ Features

-   **Premium Glassmorphic UI**: A stunning, modern design with smooth animations.
-   **Dual AI Support**: Choose between Google Gemini (Free Tier) and OpenAI GPT-4o.
-   **Academic Standards**: Supports multiple citation styles including APA 7, MLA 9, Chicago, and Harvard.
-   **No Registration Required**: All settings and API keys are stored locally in your browser.
-   **Export Options**: Copy results to clipboard or download as `.txt`.

## 🚀 Deployment

This tool is designed to be hosted for free on **GitHub Pages**.

### How to Deploy:
1.  Go to your repository settings on GitHub.
2.  Navigate to the **Pages** section in the sidebar.
3.  Under "Build and deployment" > "Source", ensure "Deploy from a branch" is selected.
4.  Select the `main` branch and `/ (root)` folder.
5.  Click **Save**. Your tool will be live at `https://imfj007.github.io/assignment-writer-tool-/`.

## 🛠️ Usage

1.  Open the settings ⚙️ icon at the bottom right.
2.  Input your **Google AI Studio Key** or **OpenAI API Key**.
3.  Enter your topic, choose your academic level and length.
4.  Click **Generate Professional Draft**.

## 🛡️ Privacy

Your API keys are never sent to any server except the AI providers themselves. They are stored only in your browser's `localStorage`.
